# Pumpkins vs. Undead

A Halloween-themed PWA browser game where you have to place pumpkins and other botanic defenses to keep the waves of undead away.

# Content

This game includes soundtrack made by Jacek 'Fragik' Rogal for SCP: Secret Laboratory.
